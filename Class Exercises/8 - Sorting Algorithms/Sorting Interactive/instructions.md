# Sorting Interactive

## Objective

Experiment with sorting behavior using the interactive visual/task page.

## What to do

- Generate arrays and run the provided sorting action.
- Observe how item order changes after each run.
- Compare behavior with different starting arrays.

## Challenge

- Try editing `script.js` to swap in a different sorting algorithm.
- Keep UI behavior unchanged while replacing algorithm logic.

## Success criteria

- The page runs correctly with no errors.
- Sorted output is correct.
- You can explain what each algorithm step is doing.
