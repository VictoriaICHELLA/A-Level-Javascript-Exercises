# Merge Sort Assess 2 (Open Book)

## Task

Complete the merge sort algorithm in `script.js` using the provided function definitions.

## Requirements

- Keep the `mergeSort(array)` function.
- Keep the separate `merge(left, right)` function.
- `sortArray()` must call `mergeSort(array)`.
- Return a fully sorted array in ascending order.

## Success checks

- Clicking **Sort Array** should output the array in ascending order.
- Works for arrays of different sizes (including empty and single-item arrays).
