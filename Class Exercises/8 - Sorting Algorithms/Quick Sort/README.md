# Quick Sort

Quick sort is a divide-and-conquer sorting algorithm.

## Steps

1. Choose a pivot.
2. Partition values into left (smaller) and right (larger).
3. Recursively sort each partition.
4. Combine results.

## Complexity

- Average time: O(n log n)
- Worst time: O(n^2)
- Space: depends on implementation/recursion depth

Use the available examples to compare partition approaches and understand the recursion flow.
