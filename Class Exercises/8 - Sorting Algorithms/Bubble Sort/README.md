# Bubble Sort

Bubble sort repeatedly compares adjacent values and swaps them when they are in the wrong order.

## Steps

1. Loop through the array.
2. Compare each pair of adjacent items.
3. Swap when left item is greater than right item.
4. Repeat passes until no swaps occur.

## Complexity

- Time: O(n^2) average/worst
- Space: O(1)
