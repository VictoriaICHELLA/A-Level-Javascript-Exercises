# Merge Sort Single Function Assess 2 (Open Book)

## Task

Complete merge sort in `script.js` using only the provided `mergeSort(array)` function definition.

## Requirements

- Keep everything inside `mergeSort(array)`.
- Any recursive call must be to `mergeSort(...)` only.
- Do not create a separate `merge()` helper function.
- Return a fully sorted array in ascending order.

## Success checks

- Clicking **Sort Array** should output the array in ascending order.
- Works for arrays of different sizes (including empty and single-item arrays).
