# Insertion Sort

Insertion sort builds a sorted section one item at a time.

## Steps

1. Start from index 1.
2. Store the current item.
3. Shift larger items in the sorted section to the right.
4. Insert the current item in the correct position.
5. Repeat for all items.

## Complexity

- Time: O(n^2) average/worst
- Space: O(1)
