# Sorting Blank Template

## Objective

Use this page to implement and test any sorting algorithm.

## What to do

- Open `script.js`.
- Replace the placeholder sorting function with your own algorithm.
- Keep the existing display and button framework.
- Test with multiple generated arrays.

## Success criteria

- Clicking **Sort Array** outputs values in ascending order.
- The algorithm works on different array sizes.
- No runtime errors appear in the console.
