# Linear Search 2D Column

This set focuses on searching a **2D array** by a specific column.

Scenario: appointments stored as rows with this format:

- `[time, name, reason, room]`

Students search by the `time` column and return the matching appointment details.

Stages:

- Linear Search 2D Column Example
- Linear Search 2D Column Easy
- Linear Search 2D Column Medium
- Linear Search 2D Column Hard
- Linear Search 2D Column Assess
