# Linear Search (9a)

Linear search checks each item one by one until it finds the target or reaches the end.

## Steps

1. Start at index 0.
2. Compare current item with target.
3. If equal, return its location.
4. Otherwise move to the next item.
5. If end is reached, return not found.

## Complexity

- Time: O(n)
- Space: O(1)
