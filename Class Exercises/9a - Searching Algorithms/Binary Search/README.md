# Binary Search (9a)

Binary search works on a sorted array by repeatedly checking the middle value.

## Steps

1. Set low and high bounds.
2. Find the middle index.
3. Compare middle value to target.
4. Move to left or right half.
5. Repeat until found or bounds cross.

## Complexity

- Time: O(log n)
- Space: O(1) iterative implementation
