# Task 2 Example - Practicing Loops

## Objective

Use `for` loops to generate repeated output and practice loop control.

## What to do

- Review the starter code in `script.js`.
- Focus on loop start, stop condition, and increment.
- Run the page and compare output to the expected sequence.
- Modify values to test how loop behaviour changes.

## Success criteria

- Loop output is correct and complete.
- No off-by-one errors.
- Code remains clear and readable.
