# Linear Search 2D Grid

This set focuses on a full linear search over a **2D grid**.

Scenario: a board game grid where each item is a piece marker.

Students search rows and columns to locate a target piece and return its coordinates.

Stages:

- Linear Search 2D Grid Example
- Linear Search 2D Grid Easy
- Linear Search 2D Grid Medium
- Linear Search 2D Grid Hard
- Linear Search 2D Grid Assess
