# Geometry Calculator Problem

## Objective

Use functions/procedures to calculate geometry values from user inputs.

## What to do

- Identify each required calculation.
- Implement or complete separate functions for each formula.
- Call the right function based on the selected calculation.
- Display a clear result message.

## Success criteria

- Calculations are correct for multiple test inputs.
- Functions are reused instead of duplicated code.
- The page runs without console errors.
