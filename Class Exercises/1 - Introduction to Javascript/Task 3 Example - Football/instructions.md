# Task 3 Example - Football

## Objective

Practice using JavaScript logic to process football-related data and show clear output.

## What to do

- Read the starter code in `script.js`.
- Identify inputs, processing steps, and expected output.
- Complete or test the logic so the result shown is correct.
- Try a few different values to confirm the behaviour.

## Success criteria

- The page runs without errors.
- Output updates correctly for test values.
- You can explain how the calculation/decision is made.
