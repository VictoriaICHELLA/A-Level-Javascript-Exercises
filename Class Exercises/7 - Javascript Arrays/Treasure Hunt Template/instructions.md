# Treasure Hunt Template

## Objective

Use arrays to build a small treasure-hunt style search task.

## What to do

- Read and understand the starter variables and data structure.
- Complete the missing array logic in `script.js`.
- Use loops/conditions to search or update values as required by the task.
- Output clear results so the user can see what was found.

## Success criteria

- The program runs without errors.
- The expected treasure/location result is shown.
- Edge cases are handled (for example: not found).
